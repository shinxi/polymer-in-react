How - React + Polymer

Term

A predix-ui component is a polymer Element that is managed by Bower as a bower component.

Integrate Flow

Integrate Predix Component -> Integrate Polymer Element -> Integrate with bower.

Sample: Integrate px-dropdown to sample-app

create-react-app --scripts-version @svmx/react-scripts sample-app

Integrate Bower

  1. Install Bower
    2. npm install -g bower
  2. Install Bower Dependencies
    3. cd sample-app
    bower init
    bower install --save Polymer@^1.9.1
    bower install --save px-dropdown

Integrate Polymer Element

  1. Include Webcomponents polyfills
  2. Load Bower component for Polymer Element
  3. Map Polymer attributes(in dash) to React properties(in camelCase)
  4. Register/Listern events for Polymer Element

V1

react-polymer + polymer-webpack-loader

V2

loadBowerComponent + addEventListener

V3

withReactWrapper(Higher-Order Component)

Wrapping Polymer elements in React automatically

Imagine if you just need


bower install --save px-dropdown
npm run create-polymer-in-react px-dropdown

It's already at ui-components-predix!

  1. Clone your fork.
    2. git clone git@github.com:shinxi/ui-components-predix.git
  2. Install
    3. 
cd ui-components-predix
npm install
bower install
  3. Install Bower Component
    4. bower install --save px-dropdown
  4. Create polymer in react
    5. npm run polymer px-dropdown
  5. Check outputs.

You can also create polymer:all in one time

    
bower install --save px-dropdown
bower install --save px-spinner
bower install --save px-progress-bar
...
npm run polymer:all
